## Persiapan

1. Clone/Pull Repository

   ```bash
      git clone https://github.com/dhedhe-setiawan/thesis-project.git
   ```

2. Instalasi Package

   ```bash
      npm install
   ```

3. Menjalankan Server Development

